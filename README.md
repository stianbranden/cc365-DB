# Prerequisites
- [NodeJS](https://nodejs.org/en/) installed
- Clone repository
- Add .env file
- Run "npm i"
- Run "npm start"


## .env file
**Must contain:**

USER = *{username to cc365 api}*

PASS = *{password to cc365 api}*

PORT = *Port for webserver*

Alternatively these parameters can be set to enviranmental variables
